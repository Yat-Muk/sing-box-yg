### 一、Sing-box-yg精裝桶小白專享一鍵五協議共存腳本（VPS專用）

* 支持五大協議：Vless-reality-vision、Vmess-ws(tls)/Argo、Hysteria-2、Tuic-v5、AnyTLS-reality

* 支持純IPV6、純IPV4、雙棧VPS，支持amd與arm架構，支持alpine系統，推薦使用最新的Ubuntu系統

* 小白簡單模式：無需域名證書，回車三次就安裝完成，複製、掃描你要的節點配置

### VPS專用一鍵腳本如下：快捷方式：```sb```

```
bash <(wget -qO- https://raw.githubusercontent.com/yat-muk/sing-box-yg/main/sb.sh)
```
或者
```
bash <(curl -Ls https://raw.githubusercontent.com/yat-muk/sing-box-yg/main/sb.sh)
```
